Delta-Epsilon Limits

An Interactive Applet powered by Sage and MathJax.

(By Kelsey Norman with HTML code from Prof. Gregory V. Bard)

\textrm{This demo produces a visual for the } \delta - \epsilon \textrm{ definition of a limit: } \\ \textrm{Let } c \textrm{ be a limit point of } D \textrm{ and let } L \textrm{ be a real number. } \\ \textrm{We say that } lim_{x \rightarrow c} f(x) = L \textrm{ if for every } \epsilon \textgreater 0 \textrm{ there exists a } \delta \\ \textrm{ such that, for all } x \in D, \textrm{ if } 0 \textless |x-c| \textless \delta, \textrm{then } |f(x) - L| < \epsilon
